## Quote Generator

A simple and professional web-based quote generator built using HTML, CSS, and JavaScript.  
The application displays random inspirational quotes from a local data source with a clean UI, smooth animations, and social sharing support.

---

## Overview

The Quote Generator randomly displays motivational and inspirational quotes along with their authors.  
It is designed as a lightweight frontend project with no external API dependency, making it fast, reliable, and easy to run locally.

---

## Features

- Random quote generation from a local dataset
- Clean and modern user interface
- Smooth animations and loading indicator
- Twitter sharing functionality
- Quote counter to track displayed quotes
- Responsive design for all screen sizes
- Automatic font adjustment for long quotes
- Fully functional without an internet connection

---

## Tech Stack

- **HTML5** – Semantic structure  
- **CSS3** – Styling, animations, responsive layout  
- **JavaScript (Vanilla)** – Application logic and DOM manipulation  
- **Font Awesome** – Icons
